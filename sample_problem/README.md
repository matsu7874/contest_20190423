# サンプル問題 ABC

## Challenge Name

ABC

## Problem Statement

3つの自然数A,B,Cが与えられます。
`A+B*C` または `A*B+C`の大きい方を出力してください。

## Input Format

```
A B C
```

## Constraints

- `0 <= A < B < C <= 100`

## Output Format

整数を一行で出力してください。

## Sample Case

### Sample Case 1

```
3 5 7
```

```
38
```

両方の式を計算すると38の方が大きいことが分かります。

- `3+5*7 = 38`
- `3*5+7 = 22`

### Sample Case 2

```
98 99 100
```

```
9998
```
